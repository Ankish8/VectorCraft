VectorCraft Help

🚀 Getting Started

  1. Upload your image by dragging and dropping or clicking to browse
  2. Choose "Standard" for most images or "Custom Palette" for color control
  3. Adjust settings in the sidebar to fine-tune quality
  4. Download your crisp, scalable SVG file

🎛️ Settings Guide

  • Quality vs Speed: Higher values = cleaner results, slower processing
  • Detail Capture: Controls how much fine detail is preserved
  • Edge Style: Sharp corners for logos, smooth curves for artwork

🎨 Custom Palette Mode

  • • Automatically suggests color palettes from your image
  • • Perfect for logos and branded content
  • • Preview different color combinations in real-time
  • • Create custom palettes with your brand colors

🔍 Vector Zoom

  • • Mouse wheel - Zoom in/out
  • • Drag - Pan around
  • • +/- keys - Zoom controls
  • • 0 - Fit to view
  • • 1 - Actual size

💡 Tips for Best Results

  • • Logos & Text: Use higher quality settings
  • • Photos: Lower detail capture works better
  • • Line Art: Sharp corner settings preserve crisp edges
  • • Illustrations: Smooth curve settings for artistic styles

📁 File Formats

Input: PNG, JPG, GIF, BMP, TIFF (max 16MB)
Output: SVG (infinitely scalable, web-ready)
